# PCB Inspection System - MATLAB

Sistema de inspeção automática para detecção de componentes faltantes/defeito em placas de circuito impresso (PCB) usando processamento de imagens no MATLAB.

## Funcionalidades

- Comparação entre PCB com defeito e template de referência
- Detecção automática de componentes faltantes
- Operações morfológicas para limpeza de ruídos
- Agrupamento de regiões sobrepostas
- Visualização com bounding boxes

## Requisitos

- MATLAB R2018b ou superior
- Computer Vision Toolbox
- Image Processing Toolbox
